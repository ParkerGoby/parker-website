---
name: new-blog-post
description: Scaffold a new MDX blog post with frontmatter
---
Create a new blog post file at src/content/blog/$ARGUMENTS.mdx with:
1. Frontmatter: title, date (today), excerpt, tags, published: false
2. Placeholder H2 sections: Introduction, Body, Conclusion
3. Run `pnpm build` to verify no TypeScript errors
